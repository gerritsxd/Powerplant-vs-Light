About The Project

The Global Power Plant Visualization Project is an interactive web-based tool designed to visualize and analyze global power plant data. By combining comprehensive power plant information with geospatial analysis, this platform enables researchers, policymakers, and the general public to explore patterns in global energy infrastructure.

Key Features

  • Comprehensive Data Visualization: Interactive map displaying global power plants with filtering capabilities
  • Energy Mix Analysis: Country-specific energy profiles showing the distribution of different power generation types
  • Statistical Tools: Advanced statistical analysis tools for exploring correlations and distributions
  • Light Pollution Correlation: Analysis of the relationship between power plants and nighttime light intensity
  • Data Export: Export capabilities for further research and analysis

Data Sources

This project utilizes the Global Power Plant Database provided by the World Resources Institute. This comprehensive dataset contains information on power plants around the world, including their geographic location, capacity, fuel type, and other attributes.

The light pollution data is based on satellite imagery of Earth at night, providing insights into human activity and energy usage patterns.

How To Use

Map View

  1. Use the filters to select power plant types
  2. Zoom in/out to explore different regions
  3. Click on markers to view detailed information
  4. Toggle between normal and light pollution layers

Statistics View

  1. Select analysis type (correlation, comparison, etc.)
  2. Choose variables to analyze
  3. Filter by region and/or fuel type
  4. Export results for further analysis

Specs View

  1. Select a country from the dropdown
  2. View energy mix visualization
  3. Examine top power plants in the table
  4. Compare with another country if desired

Data Export

  1. Navigate to the Statistics or Specs page
  2. Generate the desired visualization or table
  3. Click the export button
  4. Choose image or data export format
